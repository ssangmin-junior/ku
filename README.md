

# 🎓 KUSIS: 건국대학교 제휴 업체 통합 정보 시스템 

KUSIS는 분산된 건국대학교 제휴 업체 정보를 통합하고, 사용자 피드백 및 통계 분석을 통해 학생들의 혜택 활용도를 극대화하기 위해 구축된 웹 기반 시각화 프로젝트입니다.

## 1\. 프로젝트 개요 및 목표

### 🎯 목표 (Goal)

본 프로젝트는 제휴 혜택 확인을 위해 **인스타그램**과 **네이버 지도**를 오가야 하는 비효율성을 해소하고, 데이터 기반의 분석 결과를 통해 학생들에게 \*\*'제휴 정보를 한 번에, 빠르게 제공'\*\*하는 시스템을 구축하는 것을 목표로 합니다.

### ✨ 핵심 기능 및 성과

1.  **종합 점수 기반 추천:** 조회수, 리뷰 수, 평점을 종합한 **Rank Score** 기반의 추천 시스템 도입.
2.  **데이터 안정성 확보:** Streamlit Cloud 배포 환경에서 발생하는 모든 데이터 로드 및 인코딩 오류를 방어 로직으로 해결.
3.  **4단계 심층 탐색:** 대분류 → 소분류 요약 → 목록 → 상세 지도의 명확한 사용자 탐색 흐름 확립.
4.  **워드 클라우드:** 가게 상세 페이지에 **한글 리뷰 키워드 분석** 결과를 시각화하여 제공.

## 2\. 시스템 아키텍처 및 핵심 로직

### 📦 데이터 파이프라인 (ETL)

KUSIS는 인스타그램 단일 소스에서 혜택, 가게명, 주소 등을 추출하고, 사용자 로그를 결합하여 분석합니다.

| 파일 | 내용 | 안정화 로직 (코드) |
| :--- | :--- | :--- |
| **`data_ver2.csv`** | 메인 가게 정보 (네이버/인스타그램 정규화 데이터) | `get_absolute_path(...)`를 통한 경로 절대화 |
| **`feedback.csv`** | 사용자 리뷰 및 평점 (동적 파일) | **`encoding='utf-8-sig'`**, `skiprows=1` 적용으로 **`KeyError` 최종 해결** |
| **`Rank_Score`** | 통계 기반의 종합 점수 (추천 시스템의 핵심) | $\text{Rank Score} = (\text{평균 별점} \times 10) + (\text{총 리뷰 수} \times 1) + (\text{총 조회 수} \times 0.05)$ |

### 🌐 웹 시각화 및 UX 개선

| 개선 기능 | 적용 로직 | UX/UI 효과 |
| :--- | :--- | :--- |
| **워드 클라우드** | `font_path` 탐색 및 `fontproperties` 객체 전달 | 배포 환경에서 **한글 깨짐 문제**를 안정적으로 해결. |
| **원클릭 전환** | `st.radio` 선택 변경 감지 시 `st.session_state.page`를 변경하고 `st.rerun()`. | 목록에서 **버튼 클릭 없이** 즉시 상세 페이지로 이동 가능. |
| **인라인 정보** | 가게 목록에 **별점/리뷰 수**를 함께 표시. | 탐색 단계에서 정보 접근성 극대화. |

## 3\. 설치 및 실행 방법 (Deployment)

### ⚙️ 필수 환경 설정

이 프로젝트를 로컬 또는 클라우드에서 실행하려면 다음 패키지가 필요합니다.

```
streamlit
pandas
pydeck
wordcloud
matplotlib
```

### 🏃 Streamlit Cloud 배포 및 실행

1.  **저장소 구조 확인:** `v4.py`, `data_ver2.csv`, `feedback.csv`, `requirements.txt` (위 패키지 포함), `NanumGothic.ttf` 파일이 `/v4/` 폴더에 모두 커밋되어 있는지 확인하십시오.
2.  **배포:** Streamlit Cloud에서 해당 GitHub 저장소의 `v4.py` 파일을 지정하여 배포합니다.

**주의:** 데이터 파일은 [https://github.com/ssangmin-junior/ku/tree/main/v4](https://www.google.com/search?q=https://github.com/ssangmin-junior/ku/tree/main/v4) 경로에서 코드가 로드되도록 **`get_absolute_path`** 로직이 적용되어 있습니다.
